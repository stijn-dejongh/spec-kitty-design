# Brand

Reference for the Spec Kitty brand assets: logo, favicon, iconography guidance, and brand voice rules.

No mascot illustrations are used on any brand surface. The logo and favicon are the only permitted image assets.

---

## Iconography

Spec Kitty uses Lucide-style stroke icons throughout. These are clean, single-weight, rounded-cap line icons that read clearly at small sizes.

**Icon specifications:**

- Stroke width: 2px
- Line caps: round
- Line joins: round
- Grid: 24 × 24 px viewBox
- No filled variants; stroke only
- Never use emoji as icons
