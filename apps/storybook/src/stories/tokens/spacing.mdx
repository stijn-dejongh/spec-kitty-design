# Spacing

Reference for all layout tokens: spacing scale, border radius, shadows, and motion.

**Usage pattern:**

```css
.my-button {
  transition: background var(--sk-motion-duration-fast) var(--sk-motion-ease-out),
              box-shadow var(--sk-motion-duration-base) var(--sk-motion-ease-out);
}
```
