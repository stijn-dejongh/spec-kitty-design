## Architecture Decisions

| Decision | Choice | Rationale |
|----------|--------|-----------|
| Alpha token format | `rgba(245, 197, 24, <alpha>)` as named `--sk-color-yellow-alpha-*` tokens | Immediately actionable; CSS relative colour deferred as separate DRIFT item |
| LightMode wrapper (html-js) | `<div data-theme="light">` wrapping render output | Propagates `[data-theme="light"]` selector scope without global decorator change |
| LightMode wrapper (Angular) | Story-level `decorators` with wrapper div | Consistent with Angular Storybook patterns; no `preview.ts` global change needed |
| Demo asset fix (CI) | `sed` substitution on `href`/`src` attributes in deploy workflow | Preserves local dev paths (C-005); CI-only transformation |
| JS module type | ES module (`export function skToggleDrawer`) in `.js` file | Consumed via `<script type="module">` in demos and `import` in Storybook |
| WP03/WP04 demo file conflict | Sequence WP03 before WP04; WP04 depends on WP03 | Avoids cross-WP ownership conflict on demo HTML files |

## Risks

| Risk | Likelihood | Mitigation |
|------|-----------|------------|
| `sed` path substitution breaks CSS `url()` references in deployed demos | Medium | Scope substitution to `href`/`src` HTML attributes only; verify in CI dry-run |
| Angular `data-theme` decorator pattern conflicts with zone.js or SSR | Low | Use plain string template wrapper in story `render`, not a component class |
| CSS split causes import-order regression in Storybook | Low | Import drawer CSS after base CSS in CollapsedHamburger story; matches cascade intent |
| `<script type="module">` MIME-type rejection on some static hosts | Low | GitHub Pages serves `.js` as `application/javascript`; confirmed compatible |
